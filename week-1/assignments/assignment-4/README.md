# Nitrobolic

Explore premium supplements.

## 📝 Project Overview

**Nitrobolic** is a landing page similar to assignment-3 but instead of New Arrivals has a Special Offers section that utilizes CSS transitions and animations for an engaging product showcase.

## 🚀 Features

- **Transition Effect:** Offer card that shifts for tactile a feedback.
- **Keyframe Animation:** A pulsing status loader to indicate Limited Stock availability.
