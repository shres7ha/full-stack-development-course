# The Binge Watcher

A clean, semantic HTML5 project featuring reviews of popular television series. This project was developed as part of a Full Stack Development Course.

## 📝 Project Overview

**The Binge Watcher** is a static web page designed to showcase a series of television reviews. It utilizes modern HTML5 semantic elements to ensure accessibility and structured content delivery.

## 🚀 Features

- **Semantic HTML5:** Uses tags like `<header>`, `<main>`, `<section>`, `<article>`, and `<aside>` for better SEO and accessibility.
- **Multimedia Integration:** Includes native HTML5 `<video>` and `<audio>` players for trailers and theme songs.
- **Navigation:** Simple internal navigation linking to the reviews section.
