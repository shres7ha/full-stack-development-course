# 📝 Cinephile

Search for your favorite movies in real time.

## 📝 Project Overview

**Cinephile** is a dynamic movie discovery web application built using React.

## 🚀 Features

- **Real-Time Movie Search:** Integrates with a movie database API to deliver instant results for any title.
- **Dynamic Popular Feed:** Automatically fetches and displays trending movies upon initial load.
- **State-Driven UI:** Manages search queries and movie data efficiently using React hooks for a seamless user experience.

## 🏃 Run Locally

1. **Clone:**
   git clone https://github.com/shres7ha/full-stack-development-course.git

2. **Navigate:**
   Go to the assignment-3 folder located inside `full-stack-development-course/week-2/assignments`

3. **Install Dependencies:**
   - Ensure you have Node.js installed.
   - Open the folder in an IDE.
   - Run the command: `npm install`

4. **Configure API Key:**
   - Locate the variable for the API key inside the `.env.example` file (`VITE_TMDB_API_KEY=YOUR_TMDB_API_KEY`).
   - Replace the placeholder with your actual key from [TMDB](https://developer.themoviedb.org/reference/getting-started).
   - Rename `.env.example` to `.env`

5. **Run:**
   - Run the command: `npm run dev`
   - Open the link shown on your terminal in any modern browser (e.g., `http://localhost:5174/`).
