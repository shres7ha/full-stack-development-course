# 📝 Weather Finder

Search weather in real time.

## 📝 Project Overview

**Weather Finder** is a dynamic web application that leverages the OpenWeatherMap API to provide real-time weather data for cities worldwide.

## 🚀 Features

- **Real-Time Data Fetching:** Integrates with a third-party API to deliver instant updates on temperature, humidity, and weather conditions.
- **Error Handling & Feedback:** Includes validation for empty input and "City not found" scenarios to ensure a smooth user experience.
